Steps to get Started 1. git --version 2. open the project directory 3. git init (one time per project) 4. git config --global user.name "JonathanFernz" (one time) 5. git config --global user.email jonathanfernzofficial@gmail.com (one time) 6. git add .(adding all the changes to the staging area) 7. git reset (to unstage changes) 8. git commit -m "message" (commits the changes present in the staging area, this step has to be followed after git add) 9. git log(viewing all the commits done) 10. git status (viewing untracked branches) 11. git checkout commit_id(moves the head pointer to a particular commit, changes are reverted temporarily) 12. git checkout master(to getback to the changes reverted in previous command) 13. git remote add origin https://github.com/jfernzofficial/GIT-DEMO.git git branch -M main git push -u origin main (pushing local repository to github) 14. Adding new changes to the repository git push origin master
